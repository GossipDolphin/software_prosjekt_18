For å starte applikasjonen bare trykk på run eller shift + F10.
Main burde da åpne et nettleser vindu for Localhost 8080.
Hvis dette ikke forekommer, vennligst trykk run og manuelt skriv inn: http://localhost:8080 i URL feltet til din nettleser.
